# Rayzin

This project is a personal project to create a ray tracer, with the help
of the book [The Ray Tracer Challenge: A Test Driven Guide to Your First 3D Rendered][book].

The project will be written solely by me, however there will be AI usage related to it:

* Rider has AI built in, into intellisense and code completion.
* Claude Code will sit besides the project, but will be instructed to only point out flaws, indicate things to fix, suggest alternative ways, etc. The code will wholly be written by myself.

Rider will use AI to write my git commit messages though, I'm too lazy to be bothered to come up with good messages,
and Rider AI does a really good job of summarizing the commits.

The goal is not to built a fully fledged ray tracer for use by anyone by me, but I will eventually turn this into a nuget package.
If that provides a tool for someone else, so much the better, but the main goal here is a learning experience for me and only me.

The goal here is to use modern C# syntax and capabilities, and as appropriate, for me to learn about these things in a good way. I know *of* most
of the new syntax and capabilities, but I may not have used them much. This is that opportunity.

  [book]: https://www.amazon.com/Ray-Tracer-Challenge-Test-Driven-Renderer/dp/1680502719
